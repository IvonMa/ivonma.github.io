macOS设置同时访问内外网

发表于 2017-09-26 | 分类于 开发环境 | |

如果只需要保证外网访问：

直接在 网络-设定服务顺序 中将外网连接的优先级设置为高于内网连接即可。

注意：该设置可以保证外网访问，但内网的访问不一定能够保证。有的应用可以在该设置下正常访问内网，有的则不行，比如数据库或连接内网共享文件夹。

如果还需要确保内网访问：

网络设置中的设定，只能保证外网的访问，而内网的访问则不一定能够保证。例如数据库连接或连接共享文件夹等非浏览器使用场景是无效的，必须要单独设置路由。

如：

sudo route -n add -net 11.100.100.0 -netmask 255.255.255.0 11.100.100.254

或：

sudo route -n add -net 10.0.0.0 -netmask 255.0.0.0 11.100.100.254
