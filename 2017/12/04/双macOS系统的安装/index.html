双macOS系统的安装

发表于 2017-12-04 | 分类于 其他 | |

分区

  1. 打开磁盘工具。如果没有其他系统，那么左侧磁盘列表应该只有一个磁盘。

  2. 右击该磁盘，在菜单中点击分区按钮，如下图。

  3. 进入分区页面后，点击下侧加号，新建一个分区。随后在右侧的磁盘信息界面中设定该分区的名称、类型及大小。

  4. 点击应用，即可生成分区。

安装

  1. 到MacAppStore下载系统安装包。
  2. 下载完成后，将系统安装到上面分好的新分区即可

切换系统

方式一

在启动磁盘中设定需要启动的系统所在的磁盘分区，以后启动时即可启动该系统。

方式二

在开机时按住option键，进入启动磁盘选择界面。选择需要启动的系统所在的分区。

注意事项

  • 两个macOS都是可以访问对方所在磁盘内的文件或启动对方所在磁盘下的应用程序的，并非严格隔离。
  • 对macOS而言，已经格式化完毕且装好系统的磁盘可以动态地切出新分区。但新分区删除后只能归还到它当初来自的母分区。
  • 在归还前，要将子分区的格式抹成母分区的格式类型。
  • 多个macOS系统可以使用外置存储的同一个分区进行TimeMachine进行备份，但不构成增量备份。
