macOS系统升级后开关机界面变为英文

发表于 2017-12-03 | 分类于 其他 | |

问题

macOS升级系统后，开关机页面均莫名其妙地变成了英文，而其他页面却正常显示中文。虽然不影响使用，但对于强迫症患者来说，真的是极度不友好。

我尝试过通过在设置中切换系统语言的方式来解决，无效；也尝试过在开机时使用command+option+r+p的方式重置，也无效。最终致电Apple客服后得到解决。

诊断

  • 在终端输入date，会显示系统当前的时间以及时区，如下：

    • 正常情况下应该如上图所示，显示中文格式时间，且时区是CST。
