发表于 2017-08-15 | 分类于 前端 | |

JS的组成部分

  1. ECMAScript，JS的核心，里面固定了JS的语法和词法
    目前常用的是ES5.1版本，ES6正在慢慢地被广泛使用。
    浏览器兼容性：基本全兼容

  2. DOM(Document Object Model) ：文档对象模型，主要用来跟页面打交道
    浏览器兼容性：大部分兼容，不兼容可处理

  3. BOM(Browser Object Model) ：浏览器对象模型，主要用来跟浏览器打交道

    例子：alert()，window.navigator.userAgent，window.open()等浏览器兼容性：不同浏览器表现不同，且不兼容不能处理

ES6目前的兼容情况

目前只支持高版本浏览器。
不兼容可以通过编译工具将ES6语法转成ES5代码来解决，从而在现有环境执行。但并不是所有代码都能转换，例如const就不行。

变量

可以定义常量：使用const：const PI = Math.PI;

  • 重复定义会报错

  • 修改常量的值会报错

字符串

新增的方法

startsWith(str);

判断当前整个字符串是否以str开头。返回boolean

endsWith(str);

判断当前整个字符串是否以str结尾。返回boolean

模板字符串

使用反单引号定义`str`

数组

新增了四个方法：

  1. map() 映射：根据一定规则对元素进行映射，由一个数组获得另一个数组。
  1. reduce()汇总：可以用来做数组的求和及求平均数

这个函数的运行过程是：result一开始被赋予数组第一个元素的值，而后开始不断的迭代数组，从第二个元素开始，每一次迭代都将下一个元素的值交给item，并将return回去的值作为下一次运行的result参数。有点儿类似递归。由于item被赋予的值是从数组的第二个元素开始，因此index也是从1开始的。

  1. filter() 过滤
  1. forEach() 遍历
